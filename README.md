## 📖 Project Description

**Time Zone** is a premium, pixel-perfect e-commerce landing page designed for a luxury watch store. Built entirely with **vanilla HTML5 & CSS3** — no JavaScript frameworks, no CSS libraries — showcasing the power of clean, semantic markup and advanced CSS techniques.

The website features a stunning hero section with CSS keyframe animations, an interactive product gallery with hover effects, a video showcase area, and a fully responsive layout that adapts beautifully from desktop to mobile. Every element has been carefully designed to deliver a premium shopping experience.

> 💡 **Why this project stands out:** It demonstrates advanced front-end development skills including CSS animations, Flexbox layouts, responsive design with media queries, hover transitions, overlay effects, and a mobile-first hamburger navigation — all without a single line of JavaScript.

## 🚀 Installation Instructions

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- Git (optional, for cloning)

### Steps

**1. Clone the repository:**

```bash
git clone https://github.com/waleedtarbosh/TimeZone.git
```

**2. Navigate to the project directory:**

```bash
cd TimeZone
```

**3. Open in your browser:**

```bash
# Simply open the index.html file in your browser
# Option 1: Double-click index.html
# Option 2: Right-click → Open with → Your browser
# Option 3: Use a live server (see below)
```

> ✅ **No dependencies to install!** No `npm install`, no build tools — it's pure HTML & CSS.

## 💻 How to Run the Development Server

While you can open `index.html` directly, using a live server provides auto-reload on file changes:

### Option 1: VS Code Live Server (Recommended)

```
1. Install the "Live Server" extension in VS Code
2. Right-click on index.html
3. Select "Open with Live Server"
4. Browser opens at http://127.0.0.1:5500
```

### Option 2: Python HTTP Server

```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000
```

### Option 3: Node.js HTTP Server

```bash
npx serve .

# Then open the URL shown in terminal
```

## 🎨 Design Decisions

### 🏗️ Architecture Choices

| Decision | Rationale |
|:---|:---|
| **Pure HTML & CSS** | Demonstrates core web fundamentals without framework overhead |
| **No JavaScript** | Proves advanced interactions (hamburger menu, dropdowns, hover effects) are achievable with CSS alone |
| **Single-page Layout** | E-commerce landing pages benefit from continuous scroll storytelling |
| **Flexbox over Grid** | Better browser support and simpler alignment for this layout pattern |

### 🎨 Visual Design Philosophy

| Decision | Rationale |
|:---|:---|
| **Red & Black Theme** (`#ff2020` + `#333`) | Conveys luxury, boldness, and premium branding for watches |
| **Josefin Sans Headings** | Modern, geometric typeface perfect for luxury product presentation |
| **WebP Image Format** | ~30% smaller than PNG/JPEG with identical quality — faster load times |
| **CSS Checkbox Hack** | Mobile hamburger menu works without JavaScript using `:checked` pseudo-class |
| **Infinite Scale Animation** | Hero watch image uses a breathing/pulsing effect to draw attention |
| **Overlay Hover Effects** | Gallery overlays add depth and visual feedback for interactivity |

### 📱 Responsive Strategy

```
Desktop (≥992px)     → Full layout, all animations, side-by-side grids
Tablet (576-991px)   → Wrapped grids, collapsible nav, adjusted padding
Mobile (≤575px)      → Single column, hidden hero image, full-width cards
```
